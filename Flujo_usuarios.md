# Flujos de Usuario para la Aplicación Web de Consumo Local y Ecológico

## 1. Flujo de Usuario: Registro

1. **Pantalla de Inicio**
   - El usuario hace clic en el botón **"Registrarse"**.
   
2. **Pantalla de Registro**
   - El usuario completa el formulario con su nombre, correo electrónico y contraseña.
   - El usuario hace clic en el botón **"Registrarse"**.
   
3. **Validación**
   - Si la información es válida, se crea la cuenta y se redirige al **Dashboard**.
   - Si hay un error (por ejemplo, correo ya registrado), se muestra un mensaje de error.

---

## 2. Flujo de Usuario: Inicio de Sesión

1. **Pantalla de Inicio**
   - El usuario hace clic en el botón **"Iniciar Sesión"**.
   
2. **Pantalla de Inicio de Sesión**
   - El usuario ingresa su correo electrónico y contraseña.
   - El usuario hace clic en el botón **"Iniciar Sesión"**.
   
3. **Validación**
   - Si las credenciales son correctas, se redirige al **Dashboard**.
   - Si hay un error (por ejemplo, contraseña incorrecta), se muestra un mensaje de error.

---

## 3. Flujo de Usuario: Realizar un Pedido

1. **Pantalla de Dashboard**
   - El usuario hace clic en el botón **"Realizar Pedido"**.
   
2. **Pantalla de Productos**
   - El usuario navega por la lista de productos.
   - El usuario selecciona los productos y cantidades deseadas.
   - El usuario hace clic en el botón **"Añadir"** para cada producto.
   
3. **Pantalla de Realización de Pedidos**
   - El usuario revisa el resumen del pedido.
   - El usuario hace clic en el botón **"Confirmar Pedido"**.
   
4. **Validación**
   - Si el pedido se confirma, se muestra un mensaje de éxito y se redirige al **Dashboard**.
   - Si hay un error, se muestra un mensaje de error.

---

## 4. Flujo de Usuario: Ver Historial de Pedidos

1. **Pantalla de Dashboard**
   - El usuario hace clic en la sección **"Historial de Pedidos"**.
   
2. **Pantalla de Historial de Pedidos**
   - El usuario ve una lista de pedidos anteriores.
   - El usuario puede hacer clic en un pedido para ver más detalles.

---

## 5. Flujo de Usuario: Gestión de Cuenta

1. **Pantalla de Dashboard**
   - El usuario hace clic en el botón **"Configuración de Cuenta"**.
   
2. **Pantalla de Configuración de Cuenta**
   - El usuario actualiza su información (nombre, correo, etc.).
   - El usuario hace clic en el botón **"Actualizar Información"**.
   
3. **Validación**
   - Si la información es válida, se muestra un mensaje de éxito.
   - Si hay un error, se muestra un mensaje de error.

---

## 6. Flujo de Usuario: Soporte y Ayuda

1. **Pantalla de Dashboard**
   - El usuario hace clic en el botón **"Soporte"**.
   
2. **Pantalla de Soporte y Ayuda**
   - El usuario puede leer preguntas frecuentes.
   - El usuario puede completar un formulario de contacto para enviar una consulta.
   - El usuario hace clic en el botón **"Enviar Consulta"**.
   
3. **Confirmación**
   - Se muestra un mensaje de confirmación de que la consulta ha sido enviada.

---

## 7. Flujo de Usuario: Gestión de Usuarios (Administradores)

1. **Pantalla de Dashboard (Administradores)**
   - El administrador hace clic en **"Gestión de Usuarios"**.
   
2. **Pantalla de Gestión de Usuarios**
   - El administrador ve una lista de usuarios.
   - El administrador puede hacer clic en **"Editar"** para modificar la información de un usuario.
   - El administrador puede hacer clic en **"Eliminar"** para eliminar un usuario.

3. **Confirmación de Cambios**
   - Se muestra un mensaje de éxito tras editar o eliminar un usuario.
